- Pipe line: automated workflow (assembly line) that moves code from development to production 		by  automating build, test, and deployment stages.

DevOps job is to write these pipelines - tools (GitHub actions, GitLapCI, Jenkins)


- CI: continuous integration - auth test run when code pushed
- CD: continuous deployment - auto deploy to production when test pass

- Kubernetes: platform for automating deployment, scaling and managing containerized 			applications.
